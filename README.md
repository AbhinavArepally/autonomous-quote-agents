# autonomous-quote-agents
AI-powered insurance decision system using a multi-agent architecture. Agent 1 assesses driver risk using clustering and Random Forest, while Agent 2 predicts policy conversion probability. A decision engine combines these outputs to automatically approve, escalate, or request follow-up for insurance quotes via an interactive Streamlit app
This project is deployed here: https://autonomous-quote-agents-gitamteam19.streamlit.app/
